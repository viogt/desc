# Simple Node.js Web App

## Overview
A basic Node.js web application that serves a static HTML page using Node.js's built-in HTTP module.

## Project Structure
- `index.js` - Node.js server that serves the HTML file on port 5000
- `index.html` - Simple HTML page with basic styling
- `package.json` - Project configuration and dependencies

## Current State
The web application is running and accessible via the Replit webview. The server listens on 0.0.0.0:5000 and serves the index.html file.
